Memory Game

  • The memory game is a common children's game played with a set of cards. The cards have a pictures on one side and each picture appears on two cards.
  • The game start with all the cards face down and players take turns to turn over two cards.
  • If the two cards have the same picture, then they keep the cards otherwise they turn the cards face down again.
  • This game is built using React Js, JavaScript, HTML , and CSS.
