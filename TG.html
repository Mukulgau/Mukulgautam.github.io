Tenzies Game

  • It is a game of dice. Players get ten dice to play with.
  • Players roll dice and choose a number that are more than other number and aside them and roll again and again until they get the same number in all dice.
  • Be the first player to get all ten of your dice to show the same dice.
  • This game is build using React Js, JavaScript, CSS, and HTML.
